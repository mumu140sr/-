# 📅 シフト自動生成アプリ

焼きなまし法（Simulated Annealing）による高度な最適化エンジンを搭載したシフト作成Webアプリです。
ブラウザだけで動作し、Excel不要・サーバー不要で使えます。

## ✨ 主な機能

### 🎯 最適化エンジン
- 焼きなまし法による高度な組合せ最適化
- 複数の近傍生成戦略（個人内入替/個人間入替/シフト変更/大規模入替）
- 局所最適脱出のためのリヒート機構
- 最大20万回の反復で最適解を探索

### 📋 設定可能なルール
- **連勤上限**（可変、デフォルト4日）
- **遅→早の禁止**（インターバル不足）
- **単発休みのペナルティ**（リズムの悪い休み）
- **個人ごとの公休数の上限**
- **月またぎ**（前月末の連勤考慮）
- **スタッフごとの早番/遅番希望**
- **役職別の必要人数**（早責・遅責・早総・遅総・早・遅）

### 💼 役割タイプ
- **責任者**: 早責・遅責に入る
- **総合**: 早総・遅総に入る
- **一般**: 早・遅に入る
- **オールマイティ**: 全シフトに入れる

### 🎨 視覚的な操作
- 色分けされたシフト表（役職ごとに色が違う）
- ルール違反箇所の赤枠ハイライト
- **ドラッグ&ドロップで手動修正**（セル間でシフト交換）
- カレンダー風の希望休入力

### 📥 入出力
- Excelエクスポート（色付き、集計シート付き）
- CSVエクスポート（UTF-8 BOM付きでExcel対応）
- ブラウザのlocalStorageで設定保存/読込

### 📊 AI診断レポート
- 違反種別ごとに集計
- 違反箇所と推奨アクションを提示
- スタッフ別の勤務/休日数集計

## 🚀 使い方

1. `index.html` をブラウザで開く
2. **① 基本設定**で対象月・連勤上限などを決める
3. **② 役職マスター**で各シフトの必要人数を設定
4. **③ スタッフ管理**で各スタッフを登録（最大20人）
5. **④ 希望休入力**で各スタッフの希望休を入力
6. **⑤ 自動生成**ボタンを押す
7. **⑥ シフト表**で結果を確認、必要に応じてドラッグ&ドロップで修正
8. Excelエクスポートで保存

## 🛠 技術スタック

## 💡 元になったVBAコード

このアプリは、Excel VBAで書かれた「シフト最適化エンジン」を元に、ブラウザ上で動作するように再実装したものです。VBAでは100人分のシートを処理していましたが、本アプリは最大20人を想定して、より高機能・高速にしています。
